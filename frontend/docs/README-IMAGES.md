# Image Assets

Please add the following images to the `/public` folder:

## Required Images

### Main Product Images
- `robot-hero.png` - Main robot image for hero section (recommended: 1000x1000px, transparent background)
- `robot-showcase.png` - Robot image for product showcase with hotspots (recommended: 1200x1200px)

### Demo Images
- `demo-before.jpg` - Before cleaning image (recommended: 1920x1080px)
- `demo-after.jpg` - After cleaning image (recommended: 1920x1080px)

### SEO & Social Images
- `og-image.jpg` - Open Graph image for social sharing (required: 1200x630px)
- `twitter-image.jpg` - Twitter Card image (required: 1200x628px)
- `logo.png` - Company logo (recommended: 512x512px)

### Favicons
- `favicon.ico` - Browser favicon (32x32px)
- `favicon-16x16.png` - Small favicon
- `favicon-32x32.png` - Standard favicon
- `apple-touch-icon.png` - Apple touch icon (180x180px)
- `android-chrome-192x192.png` - Android icon
- `android-chrome-512x512.png` - Android icon large

## Image Guidelines

1. **Format**: Use PNG for logos/products (transparency), JPG for photos
2. **Optimization**: Compress all images before uploading
3. **Naming**: Use lowercase with hyphens (e.g., `robot-hero.png`)
4. **Alt Text**: Provided in components for accessibility
5. **Responsive**: Images will be automatically optimized by Next.js

## Temporary Placeholders

Until you add real images, you can use:
- https://placehold.co/ for placeholder images
- https://picsum.photos/ for random photos
- Or create simple colored backgrounds

## Brand Assets

For brand consistency:
- Primary Color: #0F172A
- Accent Color: #3B82F6
- Use consistent lighting and angles for product photos
- Maintain white/light background for product images
- Ensure high resolution (2x for retina displays)
