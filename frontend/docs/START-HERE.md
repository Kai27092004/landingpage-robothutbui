## 📝 HƯỚNG DẪN

### Bước 1: Dừng Server Hiện Tại
Trong terminal chạy `npm run dev`, nhấn:
```
Ctrl + C
```

### Bước 2: Chạy Lại Server
```bash
npm run dev
```

### Bước 3: Mở Trình Duyệt
```
http://localhost:3000
```

**Nếu vẫn chưa thấy styles:**

Hard refresh:
```
Ctrl + Shift + R  (Windows/Linux)
Cmd + Shift + R   (Mac)
```

## 🔧 Nếu Có Vấn Đề

### Module Not Found
```bash
Remove-Item -Recurse -Force node_modules
npm install
```

### CSS Không Load
```bash
Remove-Item -Recurse -Force .next
npm run dev
```

### Port 3000 Đang Sử Dụng
```bash
taskkill /F /IM node.exe
# Hoặc chạy port khác:
npm run dev -- -p 3001
```
